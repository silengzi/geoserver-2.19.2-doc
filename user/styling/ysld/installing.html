Installing the GeoServer YSLD extensionÂ¶

The YSLD extension is listed on the GeoServer download page.

To install:

  1. Download the appropriate archive from the GeoServer download page. Please verify that the version number in the filename corresponds to the version of GeoServer you are running. The file will be called geoserver-A.B.C-ysld-plugin.zip where A.B.C is the GeoServer version.

  2. Extract the contents of the archive into the WEB-INF/lib directory in GeoServer. Make sure you do not create any sub-directories during the extraction process.

  3. Restart GeoServer.

  4. To confirm successful installation, check for a new YSLD entry in the Styles editor.
