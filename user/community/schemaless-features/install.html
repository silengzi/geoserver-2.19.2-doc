Installing the Schemaless Mongo moduleÂ¶

  1. Download https://build.geoserver.org/geoserver/2.19.x/community-latest/geoserver-2.19.x-SNAPSHOT-mongodb-schemaless-plugin.zip nightly GeoServer community module.

    Warning

    Verify that the version number in the filename corresponds to the version of GeoServer you are running (for example geoserver-2.19.2-schemaless-features-plugin.zip above).

  2. Extract the contents of the archive into the WEB-INF/lib directory of the GeoServer installation.
